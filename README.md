# cyberdeck-hw

Hardware design repository for the Cyberdeck platform
by **Kali Assistant** — HyperPROBE Labs

---

## Overview

This repository contains the **hardware design files** for the Cyberdeck project, including:

* PCB schematics and layouts
* mechanical CAD models
* fabrication and manufacturing data
* supporting hardware resources

The goal is to provide a **modular, reproducible, and hackable hardware platform** for embedded systems, networking, and cybersecurity applications.
